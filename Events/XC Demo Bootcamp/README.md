# Partner XC Bootcamp

As modern apps and APIs span hybrid, multi‑cloud, and API‑first environments, your customers are facing more risk and complexity than ever. In this F5 Distributed Cloud (XC) Demo Series, you’ll learn how to guide them using four GTM‑ready demos—XC WAF, XC DDoS Mitigation, XC API Discovery & Protection, and XC Bot Defense—and walk away with real‑world examples you can put to work immediately.

## This repository contains F5 Distributed Cloud JSON files for XC Resources

> [!IMPORTANT] These example JSON files are created with the placeholder name ``change-me``, You **must** change this to a unique value such as `first-last`, or everyone will end up creating the same resource.
